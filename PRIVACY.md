# Privacy Policy for Translation & Summary

Last updated: March 2, 2026

## Data Collection

Translation & Summary does not collect, store, or transmit any personal data to our servers. 

## Data Usage

### Local Storage
- Your API key and configuration settings are stored locally in your browser using Chrome's secure storage API
- Selected text and page content are processed temporarily in memory and are not stored

### Third-Party Services
- When you use translation or summarization features, the text is sent to your configured AI API endpoint (e.g., OpenAI)
- This data transmission is direct between your browser and the AI service
- We do not intercept, store, or have access to this data
- Please refer to your AI provider's privacy policy for how they handle data

## Permissions

This extension requires the following permissions:

- **storage**: To save your API key and settings locally
- **activeTab**: To access the current page content when you click translate/summarize
- **scripting**: To inject content scripts for text selection
- **sidePanel**: To display the extension interface

## Data Security

- All settings are stored locally using Chrome's encrypted storage
- No data is transmitted to our servers
- Your API key is only used to authenticate with your chosen AI provider

## Changes to Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted in the extension's repository.

## Contact

For privacy concerns, please contact us through the GitHub repository.
