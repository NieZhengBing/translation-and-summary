## Detailed Description

Transform your browsing experience with Translation & Summary - an AI-powered Chrome extension that brings intelligent translation and summarization to your fingertips.

🌐 **Multi-Language Translation**
Instantly translate selected text or entire web pages into over 10 languages including Chinese, English, Japanese, Korean, French, German, Spanish, Russian, and Arabic. Smart language detection prevents unnecessary translations.

📝 **Intelligent Summarization**
Generate concise, AI-powered summaries of any web page with a single click. Perfect for quickly understanding long articles, research papers, or documentation.

⚙️ **Fully Customizable**
- Use your own OpenAI API key or any compatible service
- Customize translation prompts and target languages
- Configure AI models (GPT-4, GPT-4o-mini, etc.)
- Support for Azure OpenAI and other providers

🎨 **Beautiful Interface**
Modern cyberpunk-style side panel with smooth animations and intuitive controls. Access all features without leaving your current page.

🔒 **Privacy First**
- All data stays local - no third-party servers
- API keys stored securely in Chrome's encrypted storage
- Direct communication with your chosen AI provider
- No tracking or data collection
