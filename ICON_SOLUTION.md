# Firefox 图标问题终极解决方案

## 当前状态
- ✅ PNG 文件已替换（文件大小正常）
- ✅ manifest.json 配置正确
- ❌ 图标仍然不显示

## 可能的原因

### 1. Firefox 临时加载的限制
Firefox 在「临时载入附加组件」模式下，可能不会立即显示图标。这是正常的。

**解决方案**：
- 图标会在正式发布后显示
- 或者重启 Firefox 后可能显示

### 2. 图标文件格式问题
虽然文件大小正常，但可能：
- 颜色模式不对
- 透明通道有问题
- 文件头损坏

**解决方案**：
使用标准工具重新生成图标

### 3. Firefox 版本问题
某些 Firefox 版本对图标支持有限。

## 建议的做法

### 方案 A：先提交，后续修复（推荐）
1. 使用当前的 `firefox-v1.0.5.zip` 提交到 Firefox Add-ons
2. 在审核过程中，图标可能会正常显示
3. 如果审核通过但图标还是有问题，可以在下一个版本更新

### 方案 B：使用 ICO 格式
Firefox 对 ICO 格式支持更好。

1. 访问：https://convertio.co/png-ico/
2. 上传你的 PNG 文件
3. 转换为 ICO
4. 在 manifest 中使用 ICO

### 方案 C：使用 Data URI（内联图标）
将图标直接编码到 manifest 中，但这会增加文件大小。

## 测试建议

### 在 Firefox 中完整测试
1. 关闭所有 Firefox 窗口
2. 重新打开 Firefox
3. 访问 `about:debugging#/runtime/this-firefox`
4. 移除旧的临时插件
5. 重新加载 `firefox-v1.0.5.zip`
6. 检查工具栏

### 检查 Firefox 控制台
1. 按 F12 打开开发者工具
2. 切换到「控制台」标签
3. 查看是否有图标加载错误
